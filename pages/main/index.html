Task: Билеты на событие

Описание

Предоставлен пример таблицы из phpMyAdmin. Указан ряд полей с описанием.

Задачи:

- показать итоговый вид таблицы с учетом дополнительных полей (тип билета).

- показать итоговый вид таблицы с учетом дополнительных полей (при наличии уникальных баркодов у каждого билета, отобразить их в таблице).

Выполнено

Задачу можно было решить разными способами, одним из которых является создание на хостинге соответствующей БД с последующим внесением в нее информации, а так же дальнейшее ее получение и преобразование в читаемый вид.

Было использовано другое решение:

Исходная таблица с частью данных, которые выбирает пользователь создана в html. Часть данных, которая должна быть сгенерирована на сторроне сервера, передана в объекты, таблица генерируется по данным, указаным пользователем после оформления заказа.

Данные для сохранения записываются в Local storage, присутствует копка очистки "ls".

Функционал скрытия баркодов не реализовывал, так как его отсутствие не противоречит условию первой задачи.
